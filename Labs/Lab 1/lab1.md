Blockly

``` javascript
while (notDone()) {
  if (isPathLeft()) {
    turnLeft();
  }
  if (isPathForward()) {
    moveForward();
  }
  if (isPathRight()) {
    turnRight();
  }
  if (isPathForward()) {
    moveForward();
  } else {
    turnRight();
  }
}
```
